# Godot Digital Asset Collection Demo

## Overview
This project is an interactive 2D game demo built in Godot. It demonstrates basic game-engine workflows including digital asset organization, player movement, collision detection, collectible objects, UI scoring, and scene management.

The project was created as a portfolio piece to show hands-on experience with Godot and digital asset refinement workflows relevant to AI training and evaluation projects.

## Project Features
- 2D interactive scene built in Godot
- Player movement using keyboard controls
- Collectible digital assets
- Collision detection
- Score tracking UI
- Organized asset folders
- Clean project structure
- Basic GDScript logic

## Tools Used
- Godot Engine
- GDScript
- 2D Scene Editor
- GitHub for version control and documentation
